# Recon CTF

## Kurulum

1. **Projeyi klonla:**
   ```sh
   https://github.com/Ahmet-MahirDEMIRELLI/recon-ctf.git
   ```

2. **Klasöre Gir:**
   ```sh
   cd recon-ctf 
   ```

3. **Ayağa Kaldır:**
   ```sh
   docker compose up --build -d
   ```

4. **Çözmeye Başla:**
   ```sh
   http://localhost:4200/
   ```
